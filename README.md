# IPL 2022 Points Table

## Description
This project displays the IPL 2022 points table by fetching data from an API and showing it in a table format. The data is sorted in ascending order based on NRR.

## Features
- Fetch data from API
- Sort teams based on Net Run Rate (NRR)
- Display data in table format
- Dynamic rendering using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)

## API Used
https://my-json-server.typicode.com/FreSauce/json-ipl/data

## How to Run

1. Open index.html in browser
2. Data will be fetched and displayed automatically
